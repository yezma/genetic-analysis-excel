# 🧬 Genetic Analysis & Interactive Dashboard

## 📋 Project Overview
This project focuses on the end-to-end data analysis of a genetic dataset using Microsoft Excel. The goal was to transform raw, messy genomic data into a structured format, extract meaningful insights regarding genetic variants, and build an interactive dashboard for clear data storytelling.

---

## 🛠️ Data Analysis Workflow

### 1. Data Cleaning & Preprocessing
Before analysis, the raw dataset was processed to ensure accuracy and consistency:
* Handled missing or null values in critical genetic marker columns.
* Standardized formatting for genetic variants and sample IDs.
* Removed duplicate records to maintain data integrity.

### 2. Data Summarization (Pivot Tables)
Utilized **Pivot Tables** to aggregate and segment the genetic data:
* Calculated the frequency of specific gene mutations across patient cohorts.
* Segmented variants by chromosomal location and classification.
* Derived statistical summaries to pinpoint high-risk genetic variations.

### 3. Visual Highlights (Conditional Formatting)
Applied **Conditional Formatting** rules to make key insights immediately scannable:
* Highlighted critical mutation thresholds and outlier frequencies.
* Color-coded variant classifications (e.g., benign vs. pathogenic) for quick visual sorting.

### 4. Interactive Dashboard & Visualization
Built a dynamic **Excel Dashboard** to present findings visually:
* Created tailored charts (e.g., Bar charts, Pie charts) to compare mutation distributions.
* *[Optional]* Implemented **Slicers** to allow users to filter the charts dynamically by demographic or mutation type.
